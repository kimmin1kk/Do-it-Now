# [2869] 달팽이는 올라가고 싶다

---

## 📌 **Algorithm**

---

수학

## 📍 **Logic**

---

```
int day = (int) Math.ceil((v-b) / (a-b));
```

- 달팽이가 도착한다는 건, 올라가는 시점이기 때문에 올라가는 것이 x일째라고 하면 내려가는 것은 그보다 적은 x-1일째이다.
- 그래서 “a*x - b(x-1) ≥ v”라는 부등식을 적용시켜, 정리하여 “x ≥ (v-b) / (a-b)”로 두었다.
- x는 결국 정수이므로 ceil을 사용하였다.
